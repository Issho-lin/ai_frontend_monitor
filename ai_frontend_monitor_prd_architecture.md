# AI Agent 前端性能监控平台文档索引

本文件作为项目文档入口。正式需求与技术架构已拆分为两份独立文档，开发、评审和后续维护时以拆分后的文档为准。

| 文档 | 用途 |
| --- | --- |
| [产品需求文档 PRD](./ai_frontend_monitor_prd.md) | 说明产品定位、用户场景、功能范围、指标口径、页面原型、业务流程和验收标准 |
| [技术架构设计书](./ai_frontend_monitor_architecture.md) | 说明系统架构、数据链路、技术选型、Agent 架构、工具接口、数据模型、权限合规和风险降级 |

当前项目主链路：

```text
SDK 接入
  -> 数据上报
  -> 指标聚合与查询
  -> 异常告警
  -> Agent 只读取证诊断
  -> 人确认治理任务
  -> 修复发布后验证
  -> 复盘沉淀
```

实现时请先对齐 PRD 中的业务范围，再按技术架构设计书拆分模块和接口。
